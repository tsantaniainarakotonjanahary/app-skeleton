# DHIS2 app skeleton
This repository can be used as a foundation to create DHIS2 apps. 

## Install
```bash
npm install
# or
yarn install
```

## Develop
You need to log in to whatever `REACT_APP_DHIS2_BASE_URL` is set to in [.env.development](./.env.development).
```bash
npm start
# or
yarn start
```

## Build
```bash
npm run build
# or
yarn build
```
